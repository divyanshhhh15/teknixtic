# Teknixtic

Teknixtic is a web-based interview preparation platform built to help students and aspiring professionals practice interview questions, track their progress, and improve their problem-solving skills through a structured learning experience.

## 🚀 Features

* User authentication and account management
* Practice technical and aptitude questions
* Track learning progress
* Interactive and responsive user interface
* Performance monitoring and feedback
* Dashboard for managing interview preparation

## 🛠️ Tech Stack

## 🎯 Project Motivation

Preparing for interviews can be challenging without a structured approach. Teknixtic was created to provide students with a platform where they can practice consistently, monitor their improvement, and build confidence before real interviews.

## 💻 Installation

1. Clone the repository

```bash
git clone https://github.com/divyanshhhh15/teknixtic.git
```

2. Navigate to the project directory

```bash
cd teknixtic
```

3. Create a virtual environment

```bash
python -m venv venv
```

4. Activate the virtual environment

```bash
venv\Scripts\activate
```

5. Install dependencies

```bash
pip install -r requirements.txt
```

6. Run migrations

```bash
python manage.py migrate
```

7. Start the development server

```bash
python manage.py runserver
```

8. Open in browser

```text
http://127.0.0.1:8000/
```
